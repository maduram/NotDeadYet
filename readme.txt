How to run
-----------
1) Set configuration values in "Config.js" 
2) Open terminal
3) Change to the directory of this project
4) Run "CreateDB" file first with >node CreateDB.js
5) Run the project with >node app.js /OR/ >node bin/www
6) Open "http://localhost:3000/" on a browser to render the project
